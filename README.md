# Medicine Store Management System 💊

A lightweight, interactive Command Line Interface (CLI) application built with **Bash Scripting**. This tool allows users to browse a medicine inventory, select items, and receive an automated bill calculation directly in the Linux terminal.

## 🚀 Features
- **Interactive Medicine Menu:** Displays names, strengths, and prices for various medicines.
- **Automated Billing:** Uses shell arithmetic to calculate the total price based on user-defined quantities.
- **User Input Validation:** Guides the user through a selection process (1-5) to choose specific medications.
- **Terminal-Based UX:** Simple, fast, and requires no external GUI or heavy dependencies.

## 🛠️ Technologies Used
- **Scripting:** Bash (Bourne Again SHell).
- **Commands:** `read`, `if-elif` logic, and `expr` for calculations.
- **Environment:** Linux / Unix Terminal.
